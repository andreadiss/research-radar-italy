# Decision Log

## 2026-05-04 - Start from MUR/Cineca

Decision: Use MUR/Cineca as the primary MVP data source.

Reason: It is the core ministerial listing for Italian university and research calls and appears to be the main source used by Academic Jobs Italy.

## 2026-05-04 - Optimize for UX and alerts

Decision: The product should compete on user experience, readable details, filtering, summaries, and alert quality, not only on aggregation.

Reason: Aggregation alone is easy to copy and already partially solved.

## 2026-05-04 - Keep source expansion in roadmap

Decision: Add PRIN/PNRR tags early, then expand to Gazzetta, inPA, Euraxess, universities, institutes, and lab pages.

Reason: Extra-MUR opportunities are a strong differentiator.
